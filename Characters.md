# Characters

## A (주인공)

| Field | Details |
|-------|---------|
| NAME | A (이름 미정) |
| ROLES | 주인공, 원정대 리더, 조선 최고의 백양석 검사 |
| PRONOUNS | 그/그의 |
| OTHER NAMES | 독수리 요새의 생존자, 한직으로 밀려난 전직 군인 |
| PERSONALITY | 냉소적, 예리한 관찰력, 자조적 유머, 무기력 속 날카로움. 겉으로는 무심한 듯하지만 동료를 챙기는 면모 |
| MOTIVATIONS / GOALS | 1) 다솜의 미래 예측에 설득당함 ("어차피 망할 거면 마지막으로 뭐라도") 2) 엘프 배신의 진실과 마왕에 대한 복수 3) 버려진 동료들을 구원하며 자신도 구원받고자 함. "우리 같은 놈들이 살 곳을 만들기 위해" |
| INTERNAL CONFLICT | 독수리 요새에서 살아남았지만 한직으로 밀려나 버려진 느낌. 부하들을 잃은 죄책감. 국가와 시스템에 대한 깊은 불신. 능력은 여전하지만 마음의 상처로 인해 검을 들 의지를 잃음 |
| STRENGTHS | 여전히 강력한 전투 능력, 상황 판단력, 리더십, 냉정함, 생존 본능 |
| WEAKNESSES | 심리적 무기력, 트라우마, 자포자기 성향, 국가와 의무에 대한 환멸 |
| CHARACTER ARC / EVOLUTION | 한직에 밀려나 무기력한 전직 군인 → 다솜의 예측과 동료들의 구원을 통해 동기 발견 → 각 동료를 구원하며 자신도 치유됨 → 검을 드는 의미를 되찾고 진정한 리더로 재기 |
| CHEMISTRY WITH PROTAGONIST / RELATIONSHIPS | 다솜에게 논리적으로 설득당함. 아라의 밝음에 처음엔 거부감, 점차 위안받음. B와는 과거의 전우이자 죄책감의 대상. C, D를 구원하며 새로운 유대 형성 |
| ROMANTIC OBSTACLES / CHALLENGES | 해당 없음 (또는 추후 설정) |
| SHARED HISTORY | 독수리 요새 참극의 생존자. 데우스를 강제 가동시킨 후 마왕과 싸우다 의식 잃음. 깨어난 후 조정의 견제로 한직(마을 사또 같은 직책)으로 밀려남. 부하들은 흩어지고 국가는 아무것도 바꾸지 않았음 |
| ROLE IN STORY / FUNCTION | 주인공, 1인칭 시점 캐릭터, 원정대를 이끄는 구심점. 플레이어가 세계를 경험하는 창 |
| SKILLS OR RESOURCES | 백양석 검술(여전히 최고 수준), 번개 조종 마법(검을 휘둘러 번개를 내려침), 전술 지식 |
| BACKGROUND ANCHOR | 독수리 요새에서 모든 것을 잃은 날. 데우스를 강제 가동시키고 마왕과 싸우다 쓰러진 순간. 깨어났을 때 세상은 변하지 않았고, 자신은 한직으로 밀려나 있었음 |
| NOTABLE DETAILS | 1인칭 현재형 독백. 거친 말투("빌어먹을", "젠장")와 시적 표현이 공존. 전직 군인의 냉소와 예리함. 능력은 온전하지만 쓸 의지가 없음. 동료들과의 연대를 통해 점차 검을 드는 의미를 되찾음. 번개를 다루는 백양석 검사 |

## B (부하)

| Field | Details |
|-------|---------|
| NAME | B (이름 미정) |
| ROLES | 원정대원, A의 전직 부하, 전사 |
| PRONOUNS | 그/그의 |
| OTHER NAMES | 독수리 요새의 생존자, 외팔의 검사 |
| PERSONALITY | 충직했으나 배신당한 후 냉소적으로 변함. 깊은 상실감과 분노를 품고 있음. 그럼에도 A에 대한 존경은 남아있음 |
| MOTIVATIONS / GOALS | 초반: 없음 (상실과 분노에 빠져있음). A와 재회 후: 국가가 아닌 동료를 위해 싸우고자 함 |
| INTERNAL CONFLICT | 국가와 임무에 목숨 바쳐 충성했으나 돌아온 것은 철저한 외면과 버려짐. A를 구하기 위해 팔을 잃었지만 아무도 기억하지 않음. 분노와 상실감 속에서도 A를 원망할 수 없는 복잡한 감정 |
| STRENGTHS | 뛰어난 전투 기술(한 팔로도), 충성심, 생존 의지, 강인한 정신력 |
| WEAKNESSES | 외팔로 인한 전투력 제한, 국가에 대한 깊은 배신감, 분노 조절 어려움 |
| CHARACTER ARC / EVOLUTION | 상실과 분노에 빠진 전직 군인 → A와 재회하며 과거 회상 → 아라의 과거 보기로 내면 치유 → 국가가 아닌 동료를 위해 싸우는 전사로 재탄생 |
| CHEMISTRY WITH PROTAGONIST / RELATIONSHIPS | A를 존경하며 그를 업고 후퇴하다 팔을 잃음. A에 대한 복잡한 감정(존경, 죄책감, 원망 없는 상실). 아라에게 치유받음 |
| ROMANTIC OBSTACLES / CHALLENGES | 해당 없음 (또는 추후 설정) |
| SHARED HISTORY | 독수리 요새에서 A를 업고 후퇴하는 과정에서 팔 한쪽을 잃음. 1-1 대본의 "장군! 제게 업히십시...!" 장면 |
| ROLE IN STORY / FUNCTION | 첫 번째 구원 대상. A의 과거와 독수리 요새 참극을 보여주는 거울. 국가의 배신을 상징하는 캐릭터 |
| SKILLS OR RESOURCES | 백양석 검술(한 팔로도 강력함), 땅을 조종하는 마법(1-1 대본의 돌덩이 끌어올리기), 전투 경험 |
| BACKGROUND ANCHOR | A를 업고 후퇴하다 팔을 잃은 순간. 그 후 국가로부터 아무런 보상도 인정도 받지 못하고 버려진 경험 |
| NOTABLE DETAILS | 1-1 대본에서 "약한 소리 마십시오, 장군. 여기서 쓰러지시면 조선은 끝입니다." 같은 충직한 모습. 현재는 그 충성심이 배신감으로 변함. 땅을 다루는 백양석 검사 |

## C (낙방생)

| Field | Details |
|-------|---------|
| NAME | C (이름 미정) |
| ROLES | 원정대원, 만년 낙방생 |
| PRONOUNS | 그/그의 |
| OTHER NAMES | 과부의 아들, 낙방생 |
| PERSONALITY | 냉소적이고 반항적. 예리하고 똑똑하지만 시스템에 대한 환멸로 의욕을 잃음. 겉으로는 건방지지만 내면에는 깊은 좌절감과 분노 |
| MOTIVATIONS / GOALS | 초반: 없음 (무기력과 방황). A를 만난 후: "나와 함께하면 세상을 바꾸게 해주겠다"는 A의 말에 감명받아 원정대 합류. 썩은 시스템 밖에서 진짜 변화를 만들고자 함 |
| INTERNAL CONFLICT | 마왕의 침략 이후 쇄락한 조선에서 계층 이동은 불가능해졌고, 기득권은 자기 밥그릇 지키기에만 급급하며 아래 세대에게 '효'라는 이데올로기로 희생을 강요함. 관직에 올라봤자 무엇 하나 바꾸지 못하고 평범한 삶을 살다 끝날 것이라는 무기력함. 공부해야 할 동기를 찾지 못해 낙방을 반복. 어머니에 대한 죄책감과 시스템에 대한 분노 사이의 갈등 |
| STRENGTHS | 뛰어난 지적 능력, 예리한 통찰력, 반골 기질, 사회 구조에 대한 깊은 이해 |
| WEAKNESSES | 시스템에 대한 환멸로 인한 무기력, 자포자기 성향, 권위에 대한 반항심 |
| CHARACTER ARC / EVOLUTION | 좌절과 무기력에 빠진 낙방생 → A를 만나 "세상을 바꾸겠다"는 약속에 감명받음 → A와 아라에게 구원받으며 진짜 변화를 만들 수 있다는 희망 발견 → 썩은 시스템이 아닌 동료들과 함께 세상을 바꾸는 인물로 성장 |
| CHEMISTRY WITH PROTAGONIST / RELATIONSHIPS | A의 냉소적 태도와 시스템에 대한 불신에 공감하며 동질감을 느낌. A의 "세상을 바꾸겠다"는 약속에 처음으로 희망을 느낌. A에게 구원받음. 아라의 과거 보기로 내면 치유 |
| ROMANTIC OBSTACLES / CHALLENGES | 해당 없음 (또는 추후 설정) |
| SHARED HISTORY | 홀어머니(과부)의 헌신적인 뒷바라지 속에 자랐으나, 마왕 침략 이후 쇄락한 조선에서 계층 이동 불가능, 매관매직 만연, '효' 이데올로기 강요를 목도하며 깊은 회의감. 과거 시험에 연달아 낙방하고 방황 중 |
| ROLE IN STORY / FUNCTION | 두 번째 구원 대상. 조선 사회의 부패와 '효' 이데올로기의 문제를 상징. 젊은 세대의 좌절과 분노를 대변. A에게 구원받으며 A의 리더십과 변화 의지를 보여주는 캐릭터 |
| SKILLS OR RESOURCES | 뛰어난 학식과 전략적 사고, 바람을 활용하는 쾌검(백양석 검술), 각성자이지만 훈련 부족으로 미숙함, 사회 시스템에 대한 깊은 이해 |
| BACKGROUND ANCHOR | 마지막 과거 시험 낙방 후 어머니의 눈물을 본 순간. 그때 깨달은 것은 "관직에 올라도 아무것도 바꿀 수 없다"는 절망 |
| NOTABLE DETAILS | 반항적이고 냉소적인 말투. "미래 없는 나라에서 입신양명이 무슨 의미냐", "기득권은 밥그릇만 지키고 우리에게 효를 강요한다"는 질문을 던지는 인물. A의 "세상을 바꾸겠다"는 약속에 처음으로 희망을 느낌 |

## D (음악가)

| Field | Details |
|-------|---------|
| NAME | D (이름 미정) |
| ROLES | 원정대원, 비주류 음악가, 힐러 |
| PRONOUNS | 그녀/그녀의 |
| OTHER NAMES | 아웃사이더, 밴드 음악가 |
| PERSONALITY | 자유분방하고 표현적. 겉으로는 밝지만 내면에는 깊은 자괴감과 외로움. 자신의 음악이 쓸모없다고 생각하며 가치를 찾지 못함 |
| MOTIVATIONS / GOALS | 초반: 없음 (자신의 음악이 무가치하다고 생각). A를 만난 후: "네 음악은 사람을 치유한다. 우리에게 필요하다"는 말에 처음으로 자신의 가치를 발견. 동료들을 치유하고 자신의 음악이 세상에 도움이 되는 것을 보고자 함 |
| INTERNAL CONFLICT | 언더우드(엘프 나라)의 엄숙하고 보수적인 사회에서 비주류 음악(J-pop 스타일 밴드 음악)을 연주하며 억압받음. 자신의 음악이 아무 쓸모없고 세상에 필요 없다는 자괴감. 존재 가치에 대한 깊은 회의 |
| STRENGTHS | 음악을 통한 치유 능력, 자유로운 사고, 창의성, 긍정적 에너지 |
| WEAKNESSES | 자괴감, 존재 가치에 대한 회의, 전투 능력 부족, 사회적 소외감 |
| CHARACTER ARC / EVOLUTION | 자신의 음악이 무가치하다고 생각하는 음악가 → A에게 "네 음악은 사람을 치유한다. 우리에게 필요하다"는 말을 듣고 구원받음 → 아라의 과거 보기로 내면 치유 → 원정대의 힐러로서 동료들을 치유하며 자신의 가치 발견 → 자신의 음악이 세상을 바꿀 수 있다는 확신을 갖는 인물로 성장 |
| CHEMISTRY WITH PROTAGONIST / RELATIONSHIPS | A에게 구원받음. A의 "네 음악은 필요하다"는 말에 처음으로 자신의 가치를 인정받음. 아라와는 밝은 성격으로 잘 맞음. 원정대에서 음악으로 동료들의 상처를 치유하는 역할 |
| ROMANTIC OBSTACLES / CHALLENGES | 해당 없음 (또는 추후 설정) |
| SHARED HISTORY | 언더우드의 보수적 사회에서 비주류 음악을 연주하며 억압받음. 자신의 음악이 아무 쓸모없다고 생각하며 자괴감에 빠져있었음 |
| ROLE IN STORY / FUNCTION | 세 번째 구원 대상(언더우드에서). 사회의 억압과 존재 가치에 대한 회의를 상징. 원정대의 힐러 역할. A에게 구원받으며 A의 리더십과 동료를 인정하는 능력을 보여주는 캐릭터 |
| SKILLS OR RESOURCES | 음악을 통한 치유 마법, 백양석을 활용한 음파 공격(보조), 창의적 문제 해결 능력 |
| BACKGROUND ANCHOR | 언더우드에서 자신의 음악이 "쓸모없다"고 여겨지며 존재 가치를 잃어가던 순간. A를 만나 처음으로 "필요하다"는 말을 들음 |
| NOTABLE DETAILS | 밝고 자유분방한 말투. J-pop 스타일 밴드 음악 연주. 원정대의 힐러로서 음악으로 동료들의 상처를 치유함. 2막(언더우드)에서 A에게 구원받음. 여성 캐릭터 |

## 다솜 (미래를 보는 기계 정령)

| Field | Details |
|-------|---------|
| NAME | 다솜 (Dasom) |
| ROLES | 데우스의 열쇠, 기계 정령, 원정대의 전략가 |
| PRONOUNS | 그녀/그녀의 |
| OTHER NAMES | 미래를 보는 자, 양자 컴퓨터 정령 |
| PERSONALITY | 무뚝뚝하고 기계적. 논리적이고 냉정함. 감정 표현이 서툴지만 인류를 구하고자 하는 목적의식은 확고함 |
| MOTIVATIONS / GOALS | 인류의 멸망을 막는 것. 미래 예측을 통해 최적의 경로를 찾아 데우스를 가동시키고 마왕을 물리치는 것 |
| INTERNAL CONFLICT | 독수리 요새에서 완성되었지만 데우스 가동 실패. 90%의 예측도로는 완벽하지 않다는 자각. 감정을 이해하지 못하지만 동료들과의 여정을 통해 변화하는 자신을 발견 |
| STRENGTHS | 미래 예측 능력(90% 정확도), 양자 컴퓨터 수준의 연산 능력, 논리적 사고, 전략 수립 |
| WEAKNESSES | 감정 이해 부족, 무뚝뚝한 태도로 인한 소통 어려움, 독수리 요새 참극의 트라우마 |
| CHARACTER ARC / EVOLUTION | 무뚝뚝한 기계 정령 → 동료들과의 여정을 통해 감정을 조금씩 이해 → 논리뿐 아니라 감정도 이해하는 존재로 성장 → 독수리 요새에서 데우스를 성공적으로 가동 |
