# Worldbuilding

| Category | Details |
|----------|---------|
| **Setting Type** | 아모레 대륙 - 세 종족(인간, 엘프, 드워프)이 공존하는 판타지 세계. 마왕 침략 이후 쇄락한 세계 |
| **Core Atmosphere** | 겉으로는 밝고 희망찬 듯 보이지만, 이면에는 배신, 부패, 억압이 만연. 마족의 위협 속에서 각 종족은 생존을 위해 발버둥치지만 내부 갈등으로 분열됨 |
| **Physical Details** | 1) 무궁화 산맥: 마족을 막는 자연 방벽, 보라색 무궁화가 피어있음 2) 백양석: 은백색으로 빛나는 특수 광석, 마법 매개체 3) 독수리 요새: 드워프의 수도, 현재 마족 점령 4) 한양: 조선의 수도, 마키나가 협곡을 사수 5) 언더우드: 엘프의 폐쇄적 나라 6) 운석 낙하 지점: 엘프 영토, 마족 탄생의 근원 |
| **Time & Rhythm of Life** | 독수리 요새 참극 이후 몇 년. 조선은 쇄락하여 계층 이동 불가능, 매관매직 만연. 엘프는 폐쇄적으로 전통 고수. 드워프는 피난민으로 흩어짐. 마족의 위협은 계속되지만 무궁화 산맥 덕분에 직접 침략은 제한적 |
| **Tone of Narrative** | 절망 속에서도 희망을 찾는 이야기. 어두운 현실을 직시하되 연대와 구원을 통해 빛을 발견 |
| **Protagonist's Inner World** | A의 시선으로 본 세계는 냉소적이고 예리함. 겉으로 보이는 질서와 명분 뒤에 숨겨진 부패와 무기력을 꿰뚫어 봄. 하지만 동료들과의 여정을 통해 점차 희망을 되찾음 |
| **Emotional Geography** | 조선(무기력과 부패) → 언더우드(억압과 배신의 진실) → 독수리 요새(과거와의 대면, 최종 결전). 각 지역은 원정대의 감정적 여정과 연결됨 |
| **Emotional Rules of the World** | 단순한 선악 구도 없음. 엘프의 배신도 이유가 있고, 벼슬아치도 진심으로 조선을 위한다고 믿음. 시스템의 문제가 개인을 착취하는 구조 |
| **Natural Elements** | 무궁화(보라색): 마족을 막는 신비한 힘. 백양석: 마법의 매개체. 운석: 마족 탄생의 근원. 자연물(번개, 땅, 바람): 백양석 검사들이 조종하는 힘 |
| **The Soul of the World** | 완벽하지 않은 존재들의 연대. 국가나 시스템이 아닌, 개인과 개인의 유대가 세상을 구하는 진짜 힘 |
